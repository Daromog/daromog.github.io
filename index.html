About me:

I'm an Ecuadorian Engineer working on AI. Currently I'm a PhD Student in Computer Vision at MBZUAI working with Ivan Laptev. Previously I worked as a research assistant with Thamar Solorio in Vision-Language models. Before that, I worked in CV, NLP and Speech processing topics at the Universidad Politecnica Salesiana in Ecuador with Christian Salamea, in the Speech Technology and Machine Learning Group at the Universidad Politecnica de Madrid with Luis Fernando D'Haro and in the Ixa Research Group at the UPV with Eneko Agirre

Fun Other Projects


GPT+BLIP

Testing the combination of "Blip" - "GPT-3.5" and applying that to surveillance gets really interesting results to describe what's going on...

Separating silences [CODE] Part of this project : [Web-Page]

I made this program to separate silences from cough recordings. It is based on the use of spectrograms and standard deviation values. I made this program as part of a research project to detect COVID-19 using coughs.

Portraits

I made this portrait of Guayasamin (Famous Ecuadorian Painter) using an algorithm based on masks and attention centers.

Robot Arm

This project was part of my first publication. This is a robot arm made in a 3D printer, it was moved by a glove with flex sensors and a Bluetooth module, it was programmed with microcontrolers. It was a proposed as an alternative option for joystics that are used to drive industrial machines.

Drop Tower

This was a fun project to simulate the famous disney drop tower, this project used a stepper motor to move the plataform and it was programmed with a microcontroler, it was built to be activated with PLC.

Controlling a ball

This project had the goal of maintining the ball in the center of the rail. It had a sensor on one side to sense the ball position. It was controlled by a PID programmed in arduino, was also monitored from matlab.
